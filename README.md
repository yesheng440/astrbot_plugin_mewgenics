# 🐱 AstrBot-Plugin-Mewgenics

这是一个为 [AstrBot](https://github.com/Soulter/AstrBot) 开发的插件，专门用于快速查询《喵喵的结合》(Mewgenics) 的 Wiki 数据。

## 📋 使用前提 

由于《喵喵的结合》图鉴网站是一个现代化的单页应用 (SPA)，内容通过 JavaScript 动态渲染，传统的爬虫无法获取数据。因此，本插件使用 **Playwright** 模拟浏览器环境进行数据抓取。

请在安装插件前，根据你的运行环境依次完成以下配置：

###  核心安装步骤
* **第一层（Python 依赖）**：安装核心驱动库。
  ```bash
  pip install playwright
* **第二层（浏览器二进制）**：安装运行所需的 Chromium 内核。
  ```bash
  playwright install chromium
* **第三层（系统底层共享库）**：仅 Docker 用户或 Linux 用户需要执行。用于补全 Linux 运行浏览器所需的底层依赖。
  ```bash
  playwright install-deps
注意：第一步和第二步是所有环境的必选项。如果你的 Bot 运行在 Docker 容器内，请务必执行第三步，否则浏览器将无法启动。
## ✨ 插件功能

本插件通过对接 Wiki 数据，实现在聊天窗口内直接查询以下内容：

* 📦 **物品**：查询各种道具的基础属性与效果。
* 🧬 **突变**：获取猫猫突变后的数值变化。
* 🔥 **能力**：查看主动技能的具体数值与消耗。
* 🛡️ **被动**：查询被动技能的触发条件与效果。
* 👤 **角色**：查询各角色的基础数值与描述。

---

## 🎮 使用方式

在聊天窗口使用以下格式进行查询：

| 查询类别 | 指令格式 | 示例 |
| :--- | :--- | :--- |
| **查物品** | `查物品 [名称]` | `查物品 废金属帽` |
| **查突变** | `查突变 [名称]` | `查突变 Body 300: Rock Bod` |
| **查能力** | `查能力 [名称]` | `查能力 投钉` |
| **查被动** | `查被动 [名称]` | `查被动 腐败` |
| **查角色** | `查角色 [名称]` | `查角色 B 型探测机` |
